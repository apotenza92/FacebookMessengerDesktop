# System Tray Icons

Place your system tray icons here:

- `icon.ico` - Windows tray icon (16x16 or 32x32)
- `iconTemplate.png` - macOS tray icon (22x22, should be a template image with transparency)
- `icon.png` - Linux tray icon (22x22 or 24x24)

Tray icons should be small and monochrome/grayscale for best visibility.

For now, Electron will use default icons if these files don't exist.
